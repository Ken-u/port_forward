端口转发管理

通过一个主端口转发多个 TCP 连接，无需改动 frp 配置

节点

名称地址状态操作
暂无节点

节点地址填写本机 frp stcp visitor 的 bind_addr:bind_port，不是业务服务端口。

转发规则

名称本地监听目标地址拨号状态操作
暂无规则

本地监听端口建议使用 1024–65535。1–1023 为系统特权端口，普通用户启动会报权限不足。

「目标在对端网络」：本机只监听，由对端机器去连目标（跨 frp 访对端内网用这个）。「目标在本机网络」：本机监听并本机直连目标（适合监听口已通过 frp 暴露、目标就在本机/局域网）。

Token 设置

已配置的 Token 不会回显；输入新 Token 并保存可进行更换。
